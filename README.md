This is a simple C++ console program that compares genetic sequences from two different labs using a **Binary Search Tree (BST)**.

## How It Works

1. The program reads data from two input files: `lab_a_data.txt` and `lab_b_data.txt`.
2. It builds a Binary Search Tree for each lab's data.
3. It then compares the two trees to find similarities and differences.

## Output Files

When you run the program, it automatically generates the following files:

- **`common_patterns.txt`**: Shows the genetic sequences that both labs share.
- **`nc_element_bst1.txt`**: Shows unique sequences found *only* in Lab A.
- **`nc_element_bst2.txt`**: Shows unique sequences found *only* in Lab B.
- **`final_report.txt`**: A quick summary report showing the level of similarity between the two datasets.

## How to Run

If you are using VS Code, simply open the project and press **F5** (or the Run button) to compile and run the program.
